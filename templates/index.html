Arab-Andalusian Patterns Visualization Tool

  • In order to visualize and listen to the patterns, you need to select the desired group (represented with shape), family of nawabāt and nawba (represented with colors).

  • Once the pattern map is plotted, you can zoom in/out around the graph and click on each node to listen to it. Each pattern is linked to other similar patterns, spatial position has no relation with the similarity.

  • If you want to see the plotted patterns in the context of a score, you can select a specific Music Brainz ID and section of the piece with the dropdown widgets and click on "Plot on score" button.
(The loading of the score might take some minutes...)

Designed by Miguel García Casado (miguel.garciac@upf.edu), within his Master Thesis on the Music Technology Group, Universitat Pompeu Fabra, Barcelona

This tool is inspired in Sankalp Gulati's "Discovery similar phrases in Carnatic Music"

MelodyShape tool from Julián Urbano was used to compute melodic similarity between patterns

For more information about the Arab-Andalusian dataset, please refer to Dunya web page
